## func config ci

Generate a Github Workflow for function deployment

```
func config ci
```

### Options

```
  -h, --help                   help for ci
  -p, --path string            Path to the function.  Default is current directory ($FUNC_PATH)
      --workflow-name string   Use a custom workflow name (default "Func Deploy")
```
